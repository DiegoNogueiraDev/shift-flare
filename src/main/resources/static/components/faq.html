Perguntas Frequentes

O XPath-Prediction, que utiliza o modelo Hercules IA internamente, analisa a estrutura do DOM e cria um perfil único para cada elemento, considerando atributos, texto, posição relativa e contexto. Mesmo que parte dessas características mudem, o algoritmo consegue identificar o elemento com base nas demais propriedades, gerando XPaths alternativos e adaptativos.

Nossa ferramenta suporta automação em diversas tecnologias web, incluindo sites em HTML5, aplicações Angular, React, Vue.js, sistemas Sharepoint, SAP Fiori, Oracle APEX, e outras interfaces baseadas em navegador. Também oferecemos conectores específicos para sistemas legados através de APIs dedicadas.

O processo ocorre em três etapas: (1) Análise do código fonte Q2, identificando componentes, dependências e funcionalidades; (2) Transformação automatizada para a estrutura Q3, adaptando os componentes para a nova arquitetura; (3) Validação e ajustes finais, com testes automáticos e assistência humana para casos complexos.

Sim, todas as nossas ferramentas são completamente integráveis com o Azure DevOps através de extensões dedicadas. A integração permite gerenciar automações como pipelines, executar code reviews automáticos em pull requests, e acompanhar processos de migração diretamente no ambiente do Azure DevOps.
